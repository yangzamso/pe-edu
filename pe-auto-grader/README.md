# 📊 엑셀 자동채점 시스템 (v2.0)

엑셀 파일의 답안 셀 색상을 자동으로 인식하여 점수로 변환하고, 원본 엑셀 구조 그대로 결과를 저장해주는 웹 애플리케이션입니다.

## ✨ 주요 기능

- **원본 구조 유지**: 채점 결과 파일은 원본 엑셀의 레이아웃, 셀 서식 등을 그대로 유지합니다.
- **색상 인식 채점**: 답안으로 칠해진 셀의 색상(초록, 노랑, 빨강)을 인식하여 점수로 자동 변환합니다.
- **자동 합계 계산**: 
  - 객관식 합계 (O열)
  - 주관식 합계 (AE열)
  - 총점 (AF열)

## 🎨 채점 기준 (RGB)

| 색상 | 점수 (객관식) | 점수 (주관식) | RGB |
|---|---|---|---|
| **초록** | 2.5점 | 5.0점 | (182, 215, 168) |
| **노랑** | 1.25점 | 2.5점 | (255, 229, 153) |
| **빨강** | 0점 | 0점 | (234, 153, 153) |

## 🚀 사용 방법

1. **`run.bat`** 파일을 실행합니다.
2. 웹 브라우저가 열리면 **"파일 업로드"** 버튼을 클릭하여 채점할 엑셀 파일을 선택합니다.
3. 정보가 표시되면 **"채점 시작 및 결과 파일 생성"** 버튼을 클릭합니다.
4. 채점이 완료되면 **"다운로드"** 버튼을 눌러 결과 파일을 저장합니다.
